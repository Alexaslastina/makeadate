🚀 СЛЕДУЮЩИЕ ШАГИ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   1️⃣  Установите MongoDB:
       Ubuntu/WSL2: sudo apt-get install mongodb-org
       macOS: brew install mongodb-community@6.0
       Windows: https://www.mongodb.com/try/download/community

   2️⃣  Запустите MongoDB:
       Ubuntu/WSL2: sudo systemctl start mongod
       macOS: brew services start mongodb-community@6.0
       Windows: Автоматически запустится

   3️⃣  Создайте .env файл:
       cp .env.template .env
       # Отредактируйте JWT_SECRET!

   4️⃣  Проверьте подключение:
       npm run test:mongodb

   5️⃣  Запустите API:
       npm run start:api

   6️⃣  Запустите Frontend (в новом терминале):
       npm run start:frontend

   7️⃣  Откройте в браузере:
       Frontend: http://localhost:4200
       API: http://localhost:3001/api/users
